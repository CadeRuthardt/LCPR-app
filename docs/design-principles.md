

# Design Principles

## Purpose

This document defines the visual and experiential philosophy for the Le Chateau Pet Resort mobile application.

Every design decision should reinforce the feeling that Le Chateau is a luxury hospitality brand for pets.

When design tradeoffs arise, this document takes precedence over personal preference.

---

# Design Mission

The app should feel like:

- Walking into a luxury hotel lobby
- Booking a stay through the Four Seasons app
- Browsing Airbnb's most premium experiences
- Using an Apple-designed application

It should **never** feel like kennel management software.

---

# Brand Personality

The interface should communicate:

- Luxury
- Hospitality
- Trust
- Warmth
- Professionalism
- Calmness
- Confidence
- Simplicity

Avoid:

- Clutter
- Visual noise
- Dense enterprise dashboards
- Outdated skeuomorphic styling
- Excessive gradients or effects

---

# User Experience Philosophy

## Hospitality First

The app should guide guests rather than ask them to complete forms.

Use conversational language.

Examples:

- "Request a Reservation"
- "Explore the Resort"
- "Preparing for Your Visit"

Avoid operational terms such as:

- Reservation Type
- Boarding Unit
- Kennel
- Internal suite codes

---

# Visual Hierarchy

Every screen should have:

1. One primary action.
2. One clear headline.
3. Limited competing elements.
4. Plenty of whitespace.

The eye should naturally flow from top to bottom.

---

# Layout Principles

- Large margins
- Comfortable spacing
- Rounded corners
- Soft shadows
- Full-width imagery where appropriate
- Minimal borders

Favor cards and sections over complex tables.

---

# Color Usage

Primary colors:

- Black Cherry
- Goldenrod
- Ivory

Supporting colors:

- Rich Mahogany
- Olive Bark
- Graphite
- Onyx

Use color intentionally.

Color should communicate importance—not decoration.

---

# Typography

Typography should establish hierarchy before color does.

Prioritize:

- Readability
- Elegant spacing
- Clear hierarchy

Avoid excessive font weights or decorative typography.

---

# Photography

Photography should emphasize:

- Happy pets
- Luxury accommodations
- Clean environments
- Genuine staff interactions
- Natural lighting

Avoid stock imagery whenever possible.

Real Le Chateau photography should become a competitive advantage.

---

# Motion

Animations should feel natural.

Use:

- Fade transitions
- Smooth page navigation
- Subtle scaling
- Gentle elevation
- Skeleton loading

Avoid:

- Flashy effects
- Bouncing animations
- Unnecessary motion

Animation should reinforce polish, not attract attention.

---

# Content Strategy

Write as a hospitality company.

Examples:

Good:

- Preparing for Your Stay
- Welcome Back
- Your Pet's Vacation
- Explore Our Resort

Avoid:

- Reservation Queue
- Boarding Request
- Kennel Assignment

Clients should never see internal operational terminology.

---

# Accessibility

Every interface should:

- Support Dynamic Type where practical
- Meet accessible color contrast
- Include descriptive labels
- Support screen readers
- Have touch targets appropriate for mobile

Luxury includes accessibility.

---

# Decision Framework

Before approving a design, ask:

1. Does this feel premium?
2. Would a luxury hotel present this this way?
3. Is the interface calm?
4. Does this reduce cognitive load?
5. Is the primary action obvious?
6. Does this reinforce the Le Chateau brand?

If the answer to any is "no," iterate before implementation.

---

# Non-Negotiable Principles

1. Hospitality over software.
2. Simplicity over feature density.
3. Brand consistency over novelty.
4. Reusable design patterns over one-off layouts.
5. Calm interfaces over busy dashboards.
6. Mobile-first design always.
7. Every interaction should leave the client feeling cared for.
