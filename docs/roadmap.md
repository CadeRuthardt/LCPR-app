

# Product Roadmap

# Le Chateau Pet Resort Mobile App

Status: Living Document

---

# Guiding Philosophy

The roadmap is organized around delivering value to clients while maintaining Gingr as the operational source of truth.

Each phase should produce a polished, complete experience—not just a collection of features.

---

# Phase 0 — Foundation

## Objective

Build the technical and design foundation.

### Deliverables

- Expo + React Native project setup
- Design system
- Theme tokens
- Navigation architecture
- Component library
- Backend architecture
- API architecture
- Product documentation
- Authentication strategy

**Success Criteria**

- Stable project structure
- Reusable UI foundation
- Clear architectural boundaries

---

# Phase 1 — MVP

## Objective

Launch a premium client companion app.

### Home
- Personalized dashboard
- Upcoming stay summary
- Current stay summary
- Vaccination reminders
- Featured content

### Pets
- Read-only Gingr pet profiles
- Vaccination status
- Feeding instructions
- Medication instructions
- Stay history

### Stays
- Current stays
- Upcoming stays
- Past stays
- Reservation requests
- Request status tracking

### Explore
- Accommodations
- Daycare
- Spa
- Cat Resort
- VIP Experience
- About
- FAQs
- Policies
- Locations

### Profile
- Account
- Notification preferences
- Family members
- Documents

**Success Criteria**

- Clients can replace the Gingr portal for everyday viewing.
- Reservation requests flow through LCPR.

---

# Deferred Until Gingr Supports Write APIs

The following are intentionally deferred:

- Editing pet profiles
- Editing feeding instructions
- Editing medication instructions
- Direct reservation creation
- Reservation modifications
- Payment management

These remain Gingr responsibilities.

---

# Release Principles

Every release should:

- Feel complete
- Improve the luxury experience
- Reduce friction
- Avoid duplicate operational data
- Keep Gingr as the source of truth

Never ship features that compromise long-term architecture simply to accelerate delivery.

---

# Product North Star

The long-term vision is for Le Chateau clients to think of this as the Le Chateau experience—not a portal into Gingr.

The app should become the digital extension of every stay, every pet, and every relationship the client has with Le Chateau.
